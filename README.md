📝 Registro de Libros en Solana
Un CRUD on-chain para gestionar tu biblioteca personal en la blockchain de Solana. Construido con Rust + Anchor desde Solana Playground.

🔧 ¿Qué puedes hacer?
| Instrucción      | Descripción |
|-----------------|------------|
| crear_usuario   | Crea tu cuenta personal (PDA) en la blockchain |
| agregar_libro   | Agrega un nuevo libro con título y autor |
| marcar_leido    | Marca un libro como leído |
| eliminar_libro  | Elimina un libro por índice |

🚀 ¿Cómo usarlo en Solana Playground?
📥 Importa el proyecto
Ve a https://beta.solpg.io/ y pega la URL de este repositorio.

🔗 Conéctate a Devnet
Haz clic en "Not Connected" → Continue. Se creará una wallet automáticamente.

🔨 Haz Build
Haz clic en el botón Build (ícono de martillo 🔨). Espera a que compile.

🚀 Haz Deploy
Haz clic en Deploy. Esto sube tu programa a la Devnet de Solana.

🧪 Corre los Tests
Haz clic en el icono de pruebas y ejecuta los tests en orden.

📂 Estructura del proyecto
lib.rs → Smart contract en Rust  
client.ts → Cliente para interactuar con la blockchain  
tests.ts → Pruebas del sistema  

📌 Notas
Este proyecto es una implementación básica para aprender el uso de Solana y Anchor, enfocándose en operaciones CRUD dentro de la blockchain.
